# GA-based Improvement Recommendations

## 목표

- 내부 링크 최적화로 체류시간 및 조회수 증대
- 고체류/저조회 콘텐츠 증폭(프로모션 대상 선정)
- 저체류 페이지(특히 `/tools/`)의 UX/콘텐츠 개선

## 자동 생성 결과 위치

- 내부 링크 제안: `src/data/internal-links-suggestions.json`
- 프로모션 후보 CSV: `src/data/ga-promotion.csv`

## 툴 페이지 개선 체크리스트

- [ ] 각 툴 페이지에 명확한 설명(문제 해결 방법)을 상단에 배치
- [ ] 예시 입력 및 결과 스니펫 추가
- [ ] CTA(다른 관련 도구/기사 링크) 추가하여 내부 이동 유도
- [ ] 모바일에서의 레이아웃/입력 방식 검토
- [ ] 성능: 불필요한 JS 제거, 서버사이드 렌더링 고려

## 프로모션 가이드라인

- 상위 프로모션 후보(스크립트 출력)을 먼저 SNS·뉴스레터에 노출
- 타겟: 체류가 높지만 노출이 적은 항목(engagement >= 60s & views <=5)
- A/B 테스트: 제목·요약·이미지 중 하나씩 변경해 효과 비교
