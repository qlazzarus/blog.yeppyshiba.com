Archive 원칙:

- 완료된 PoC의 결론, 통과 기준, artifact 위치만 남긴다.
- 현재 구현 의사결정은 `docs/chicken_farm/README.md`, `chicken_farm_next_priority_plan.md`, active PoC 문서를 우선한다.

- 파일 이동으로 링크가 깨질 수 있는 문서는 바로 옮기지 않고, 먼저 archive summary를 만든 뒤 active 문서에서 참조를 줄인다.
