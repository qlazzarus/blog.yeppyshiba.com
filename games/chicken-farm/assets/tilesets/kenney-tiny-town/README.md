# Kenney Tiny Town Tileset

PoC 1 uses Kenney's Tiny Town tileset as a temporary open asset for a chicken-farm-like 2D Tilemap.

- Source: https://kenney.nl/assets/tiny-town
- Author: Kenney
- License: Creative Commons Zero, CC0
- Tile size: 16 x 16
- Pack version noted on source page: 1.1

The copied files are:

- `tilemap_packed.png`: packed 12 x 11 tilesheet for Phaser loading
- `tilemap.png`: original spaced tilesheet from the pack
- `Tilesheet.txt`: original tilesheet metadata
- `LICENSE.txt`: original license text from the pack

Attribution is appreciated by Kenney but not mandatory under CC0.
